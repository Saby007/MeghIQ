## ✅ Overview  
This repository serves as a **central hub** for the **SfMC India team** to build and share projects that support **Mission Critical (MC) workloads** and provide **reusable components** for demos and solution accelerators.  

Our goals:  
- 🚀 Enable rapid prototyping for customer demos  
- 🔄 Standardize reusable components for MC workloads  
- 🤖 Showcase advanced capabilities like AI, DevSecOps, and automation  
- 🤝 Promote collaboration across SfMC India team

## ✨ Key Features  
- **Mission Critical Patterns**: HA/DR, geo-redundancy, failover automation  
- **Reusable Components**:  
  - Infrastructure as Code: ARM, Bicep, Terraform  
  - CI/CD Pipelines: GitHub Actions with security and compliance checks  
  - AI Integrations: Semantic Kernel, Autogen, Azure OpenAI demos  
- **Observability**: Logging, monitoring, and alerting templates  
- **Security**: Zero Trust patterns, Defender integration
